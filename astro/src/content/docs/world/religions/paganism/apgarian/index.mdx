*Apgarian Paganism* is a collective term for a group of paganist beliefs of the
people of [Apgar](/world/locations/apgar/)

# Pantheon

| Portfolio               | Name     |
| ----------------------- | -------- |
| Goddess of Civilization | Minerva  |
| Goddess of Justice      | Justicia |
| Goddess of Nature       | Pomona   |
| Goddess of Delight      | Volupta  |
| Goddess of Humility     | Amarina  |
